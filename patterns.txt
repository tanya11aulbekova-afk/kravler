# Свои regex-шаблоны для команды extract ("символы").
# Формат:  имя = регулярка       например:   promo = [A-Z]{3,}-\d{4}-[A-Z]+
# Либо просто регулярка на строке (тогда имя будет p1, p2, ...).
# Строки с # игнорируются. Эти шаблоны добавляются К встроенным (--type).
#
# Встроенные шаблоны (через --type): email, tg, tme, url, phone, ip, btc, eth, trx, code, hashtag
#
# Примеры (раскомментируй нужное):
# promo = [A-Z]{3,}-\d{2,4}-[A-Z0-9]+
# steamkey = [A-Z0-9]{5}-[A-Z0-9]{5}-[A-Z0-9]{5}
